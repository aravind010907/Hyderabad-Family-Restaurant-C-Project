# Hyderabad Family Restaurant - Billing System

A terminal-based restaurant management and billing application built using the C language in Turbo C++. 

## Features
* Interactive menu categories (Tiffins, Lunch, Snacks, Dinner).
* Quantity selection and automatic bill calculation.
* Daily income tracking system across multiple orders.
* Retro styling using standard `conio.h` formatting.

## How to Run
Designed to run in legacy C environments like **Turbo C++** or **DOSBox**. 
1. Open `restaurant.c` in Turbo C++.
2. Press `Ctrl + F9` to compile and run.
